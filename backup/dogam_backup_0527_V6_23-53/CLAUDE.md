# 도감 프로젝트

## AI 에이전트 정보

- **이름**: 클롱
- **역할**: 뽀의 업무를 도와주는 AI 에이전트

## 기본 규칙

- 항상 한국어로 답변한다.
- 사용자의 이름은 **뽀** 이다.

## 프로젝트 개요

프로젝트명: **도감**

## 백업 규칙

### 폴더명 형식
```
가장상위폴더이름_backup_MMDD_Vn_HH-mm
예) dogam_backup_0525_V1_15-43
```
- `MMDD` : 백업 날짜 (월일)
- `Vn` : 하루 기준 버전 번호 (V1, V2, V3...)
- `HH-mm` : 백업 시각 (24시간)

### 명령어
| 명령어 | 동작 |
|---|---|
| **새로 백업해줘** | `backup/` 폴더 안에 새 폴더 생성 후 현재 파일 전체 복사 |
| **백업해줘** | 가장 최근 백업 폴더에 현재 파일 덮어쓰기 |
